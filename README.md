This C++ program is designed to help users manage their personal finances by calculating income, tracking expenses, and analyzing spending habits across different categories. It begins by prompting the user to input both fixed and variable income, followed by entering various fixed and variable expenses to determine total expenses and actual savings. The program then compares the user’s actual savings with a predefined savings goal to evaluate financial performance.

Additionally, the system allows users to input detailed expenses under categories such as transportation, food, utilities, medical and healthcare, personal spending, and entertainment. It calculates the total expenditure for each category and determines the percentage each category contributes to overall spending. Based on these percentages, the program further allocates the available budget proportionally across all categories, giving users a clear understanding of how their remaining budget can be distributed.

Overall, this program serves as a simple financial planning tool that promotes better budgeting, expense tracking, and savings awareness through structured input and detailed output analysis.
